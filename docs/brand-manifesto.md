# Brand Manifesto

Pasta Lab & Smoked Flavors is a Tallinn bistro with a clear point of view: handmade fresh pasta shaped by smoke, fermentation, and Baltic character.

## Core Identity

The brand occupies a warm editorial space between craft kitchen and intimate dinner room. It feels precise, sensory, and modern, but never cold.

## Primary Payoff

Fresh pasta. Smoke. Ferment. Fire.

## Secondary Line

Italian craft, Baltic character.

## Positioning

This is a contemporary pasta bistro in a former industrial setting near the port of Tallinn. It is not a trattoria, not generic fusion, and not luxury theater. It is a focused place where Italian technique meets Northern ingredients and slow-worked flavor.

## Not This / Yes This

Not this:
- tourist-brochure Italian
- folkloric rustic
- generic luxury
- random fusion
- loud branding

Yes this:
- handmade pasta with clear craft signals
- smoked butter, fermentation, rye, dill, mushrooms, charred onion
- soft amber light
- short curated menu
- quiet confidence
- memory transformed into character

## Usage

Designers, developers, and copywriters should treat this as a guardrail document. If a new page, component, or line of copy does not reinforce the core identity, it is probably off-brand.
