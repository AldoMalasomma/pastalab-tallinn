# Project Docs

This folder is the working documentation for **Pasta Lab & Smoked Flavors**.

Current snapshot:
- Astro 6 site with Tailwind CSS v4
- Multilingual hospitality content for `it`, `en`, `et`, and `ru`
- Locale-prefixed routes for the restaurant experience in Tallinn
- Shared content source in `src/data/home.ts` and language UI helpers in `src/i18n/ui.ts`
- Shared editorial visual frames and image-slot guidance for the homepage, menu, and booking pages
- A multilingual House page that explains the founders, room logic, and hospitality rituals
